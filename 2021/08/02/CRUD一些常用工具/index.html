    • 重写hashCode 时，使用 Objects.hash(Object ...) 计算所有对象 HashCode
    • 使用 Objects.requireNonNull(Object) 代替if(Object == null) throw new NullPointerException()

  2. 集合和字符串判空常用方法

    • CollectionUtils.isEmpty 这是spring提供的工具类

    Optinal 简单用法

    • 处理允许为 null 的返回值

    • 处理不允许为null的返回值

    • 使用默认值代替null

    • 使用自定义异常对于 null

    • 过滤值

    • 使用 map 将 值重新映射为 Optional类型

    • Optional 注意事项

      • 容器类型不应该使用 Optinal
